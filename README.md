# csharp-okf

OKF v0.1 knowledge pack for C# and .NET 8+.

## Install

Point your OKF-compatible viewer at this repository. Entry point: [`index.md`](index.md). Manifest: [`okf.json`](okf.json).

## Contents

50 concept documents in [`concepts/`](concepts/) covering language fundamentals, OOP, generics, collections, LINQ, async, runtime/memory, exceptions, and ecosystem tooling.
